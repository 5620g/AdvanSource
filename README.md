# <p align="left">Clone Source:
```sh
cd $home
git clone https://github.com/janlou/AdvanSource
```
# <p align="left">First install:
```sh
cd AdvanSource
cd .run
chmod 777 adv.sh
./adv.sh r
./1.sh r
./2.sh r
./3.sh r
```
# <p align="left">install bot:
```
cd ..
```
***
```bash
chmod 777 chmod.sh
./chmod.sh r
```
***
`./start.sh install`
***
```bash
tmux new-session -s script "bash steady.sh -t"
```
